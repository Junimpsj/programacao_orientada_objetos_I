# Exemplos de GUI com Swing

Exemplos avulsos de interface gráfica usando Java Swing, cada um isolado num arquivo.

## Estrutura

```
exemplosGUIswing/
├── ExemploBorderLayout.java
├── ExemploBounds.java
├── ExemploFlowLayout.java
├── ExemploJDialog.java
├── ExemploMascara.java
├── ExemploPanelBorder.java
├── FormularioCadastro1.java
├── InputDialogGUI.java
├── JanelaCampoTexto.java
├── SomaComConfirmacao.java
├── TelaSimples.java
└── TelaSimples2.java
```

## Como executar

```bash
javac NomeDoArquivo.java
java NomeDoArquivo
```

## Requisitos

- JDK 8+ com suporte a Swing (padrão no JDK)
