# Trabalho 1: Introdução à Linguagem Java

Primeiro contato com Java: sintaxe básica, entrada/saída e operações simples.

## Estrutura

```
trabalho01_introJava/
└── Calculadora.java   soma, subtração, multiplicação e divisão via terminal
```

## Como executar

```bash
javac Calculadora.java
java Calculadora
```

## Requisitos

- JDK 8+
