# Trabalho 4: Manipulação de Arquivos em Java

Exercício de leitura e escrita de arquivos em Java.

## Estrutura

```
trabalho04_arquivosJava/
└── GerenciadorArquivos.java   operações de leitura, escrita e organização de arquivos
```

## Como executar

```bash
javac GerenciadorArquivos.java
java GerenciadorArquivos
```

## Requisitos

- JDK 8+
