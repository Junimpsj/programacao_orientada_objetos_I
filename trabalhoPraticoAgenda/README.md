# Trabalho Prático: Agenda

Aplicação de agenda de compromissos com validação de datas.

## Estrutura

```
trabalhoPraticoAgenda/
├── src/br/com/unesp/agendapp/
│   ├── aplicacao/Agenda.java     ponto de entrada
│   ├── modelo/                    Compromisso, Data
│   └── util/ValidaData.java       validação de datas
└── docs/                          javadoc gerado
```

## Como executar

```bash
cd src
javac br/com/unesp/agendapp/aplicacao/Agenda.java
java br.com.unesp.agendapp.aplicacao.Agenda
```

## Requisitos

- JDK 8+
